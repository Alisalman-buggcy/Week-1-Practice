# Day 1 and 2 
Git workflow uses feature branches, pull requests, and code reviews for safe collaboration.

TypeScript basics include types, interfaces, and type aliases for defining data structures (structural typing).

Generics make reusable and type-safe code, while utility types (Partial, Pick, Omit, etc.) simplify type transformations.

Strict mode in tsconfig adds extra checks to catch errors early and improve code safety.
